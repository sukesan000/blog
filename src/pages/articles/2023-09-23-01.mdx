## GitHub Pages とは何ぞや？

当ブログでは Github Pages を使用しているが、astro の公式ページのデプロイ方法をそのまま真似ただけなので、ドキュメントを読んで情報を整理してみた。

# GitHub Pages について

GitHub Pages は GitHub 上のリポジトリにある HTML や CSS、JS ファイルをビルドし web 上に自動でホスティングしてくれるサービスのこと。Vercel や Netlify なども同じサービス。

# どのタイミングで公開される？

変更した内容が GitHub 上の特定のブランチにプッシュされた際に公開される。

# 有料？無料？

無料プランの場合はリポジトリをパブリックに設定する必要がある。有料プランはプライベートなリポジトリで GitHub Pages を利用できるみたい。

# 使ってみた感想

初期設定さえすれば、ローカルで書いた記事をプッシュするだけで web 上に公開されるので非常に便利。Heroku みたいに途中で有料化しないように願うばかり...
